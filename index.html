Data Description

The main goal of this project is to analyze the Cost of Living in various cities around the world. By examining the data from Kaggle and adding additional geographic information, to identify trends and patterns in living expenses.

Key modifications:

  • Normalization: Cleaned the data and deleted not unusable rows Python.
  • Geographical Coordinates: Generated latitude and longitude codes for each city.
  • Continent Names: Added continent names for every entry.

Projection Explorer Map Chart

The projection map chart displays cities as triangles where color indicates average net salary and size represents the cost of a McMeal. A dropdown menu allows focus on specific continents for detailed cost of living comparisons.

Pie Chart of Country Counts grouped by Continent

This pie chart illustrates the distribution of countries across continents, providing a clear visual representation of how many countries are situated in each continent.

Packed Bubble Chart of City Counts by Country

This packed bubble chart represents each country with a bubble, where the size corresponds to the number of cities. The chart includes interactive features such as gravity in both X and Y axes, allowing bubbles to be moved for clearer comparisons.

Lasagna Plot Comparing Farm Produce Prices Across Cities

This lasagna plot features a dropdown menu to select specific countries, comparing prices of farm produce (apples, bananas, lettuce, onions, oranges, potatoes, and tomatoes) across cities within the chosen country.

Stacked Bar Chart of Market Item Prices by Country

This stacked bar chart includes a dropdown menu to select specific countries, showcasing the total prices of market items (cheese, chicken, eggs, water, bread, milk, rice, round beef).

Bubble Plot Comparing Price Ranges of Domestic and Imported Beer

This bubble plot compares the price ranges of domestic and imported beer across countries. Each bubble's size corresponds to the number of countries where that price range is observed, providing a visual comparison of beer prices globally.

Paintbrush Highlight Scatterplot Comparing Gasoline Prices and Transportation Costs

This paintbrush highlight scatterplot allows users to select and compare the price of gasoline with various transportation fees, including bus ticket prices, monthly pass prices, start price for a taxi ride, cost per km for a taxi ride, hourly waiting price for a taxi, price for a new Volkswagen Golf and the price of a brand new Toyota Corolla Sedan, using a dropdown list.

Box Plot Showing Coke Bottle Prices Across Countries

This box plot displays the maximum, minimum, and average prices of a coke bottle across different countries, providing a comparative view of pricing variability.

Summary of Global Cost of Living Data Analysis

This project analyzed global cost of living data using advanced visualization techniques. We looked at how living expenses vary across cities worldwide, using a dataset that included geographic locations and economic data.

The visualizations showed trends in affordability and factors that affect costs, which are important for making decisions in economics, city planning, and international business. In the future, adding predictive models and making the visualizations more interactive could make it easier for different people to understand and use the information.
